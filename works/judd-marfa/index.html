Overview
ミニマリズムの先駆者ドナルド・ジャッド（1928–1994）が1970年代にニューヨークを離れ、テキサス州マーファに構築した理想の芸術環境に焦点を当てた展覧会。1950年代の初期絵画、1960–90年代の立体作品、ドローイング・図面・映像資料を通じて「展示は恒久的でなければならない」というジャッドの思想を辿る。ワタリウムの創設者・和多利志津子による1978年の日本初招聘を記録したドキュメントコーナーも設置。
Source: 公式サイト / 美術手帖 / artscape — 2026.02

02 / Insight
01
素材と光の関係を体で学ぶ
アルミニウム、鉄、プレキシグラス。ジャッドの強さは余白の大きさではなく、寸法の厳密さ・素材の反復・配置の必然性にある。光がどこに当たり、どう反射するかが作品の意味そのものになっている空間を、フォトグラファーとして体で経験できる。
素材 / 光 / 撮影視点
02
「恒久展示」という思想の現場
一時的な展示ではなく「場所」を作ることがジャッドの信念だった。初期絵画から立体作品への変遷と、マーファで実践されたパーマネント・インスタレーションの思想が、ワタリウムの3フロアで一望できる。展示の「一時性」と「恒久性」の設計判断は、プロダクトやブランドの寿命設計と同じ構造を持っている。
展示設計 / 寿命設計 / 空間体験
03
日本との48年の文脈
1978年、ギャルリー・ワタリが日本初招聘で「ジャッド展」を開催。その資料が本展でも展示される。ギャラリーとアーティストが48年かけて築いた関係が一つの展覧会に結実する過程は、長期的なコラボレーションのアーカイブとしても読める。
アーカイブ / 関係構築 / 文脈
04
建築のモジュールを観察する
マリオ・ボッタ設計のワタリウム自体が彫刻的建築で、外壁の御影石ストライプはジャッドの反復構造と共鳴する。コンクリート・石・ガラスの素材選択とモジュールの繰り返しは、建築とプロダクト双方に通じるデザイン原則の実例として観察できる。
建築 / モジュール / 素材選択
